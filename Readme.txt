Chaliye mai aapke liye QuickBazaar study project ka seed version ready karta hoon.

Isme aapko milega:

HTML pages – Home, Product, Cart, Login, Signup

CSS – Simple, clean styling

JS (script.js) – Firebase integration, login/signup, product listing, cart

Firebase setup instructions – Firestore collections, Auth, Hosting

Ready-to-host link – Free Firebase Hosting

Step 1: Firebase Setup

Create Firebase Project

Go to Firebase Console
 → Add Project → Name: QuickBazaar

Enable Firestore Database

Create collections: products, users, orders, cart

Enable Authentication

Sign-in method → Email/Password

Enable Hosting

Follow Firebase Hosting setup → connect with local project
